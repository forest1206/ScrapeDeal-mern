## Prerequisites

You have nodejs, npm, mongodb installed.

## Installation

### Run backend api

```
cd/backend
npm install
npm run data:import (seed database)
npm start

```

### Run frontend

```
cd/frontend
npm install
npm start
```

Visit [localhost:3000](http://localhost:3000)
